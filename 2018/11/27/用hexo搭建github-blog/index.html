问题： hexo搭建博客过程中出现的问题，4000端口被占用

解决方案1:
先netstat -ant |grep 4000 看看被什么占用了,然后如果这个是没人用的东西,就kill掉这个进程

解决方案2:
可以在_config.yml内加上如下代码更改hexo-server运行时的端口号：
server:
port: 4001
compress: true
header: true
其中4001可以换成想要运行的端口号。
